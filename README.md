# Медицинский Telegram-бот с интеграцией 1С

Telegram-бот для медицинских поставок в Узбекистане. Клиенты — госучреждения и частные клиники. Источник правды — 1С:Предприятие.

## С чего начать

**Откройте файл [START_HERE.md](START_HERE.md)** — там подробная пошаговая инструкция от нулевого состояния до работающего бота.

## Что в проекте

- Регистрация клиентов по ИНН с проверкой в 1С
- Уровни клиентов (VIP / Стандартный / Ограниченный) с лимитами на заказы
- Каталог товаров автоматически синхронизируется из 1С
- Заказ → 1С → счёт + договор (PDF) → клиент
- Оплата по реквизитам (банковский перевод)
- Автоподтверждение оплаты через опрос 1С
- **Mock 1С** — встроенная заглушка для разработки без реальной 1С

## Технологии

aiogram 3.13 · SQLAlchemy 2 · PostgreSQL 16 · Redis 7 · httpx · FastAPI (для мока) · Docker Compose
